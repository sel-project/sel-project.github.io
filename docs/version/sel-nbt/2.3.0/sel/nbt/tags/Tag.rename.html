    Function Tag.rename

    Creates a NamedTag maintaing the tag's properties.

    abstract Tag rename (
      string
    ) pure nothrow @safe;

    Example

    auto t = new Float(22);
auto named = t.rename("float");
assert(cast(NamedTag)named);
assert(named.name == "float");
assert(named == 22);
