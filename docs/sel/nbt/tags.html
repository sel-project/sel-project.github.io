  • Named

    Declaration

    class Named(T : Tag): T;

    Creates a named tag. The first argument of the constructor becomes a string (the name) and rest doesn't change.

    Example:

    1. auto tag = new Named!Int("name", 12); assert(tag.named); assert(tag.name == "name"); assert(tag.value == 12); assert(tag == new Int(12));

  • SimpleTag

    Declaration

    class SimpleTag(T, NBT_TYPE _type): Tag;

    Simple tag with a value of type T, if T is a primitive type or it can be written in the buffer.

    Example:

    1. assert(new Short(1) == 1); assert(new SimpleTag!(char, 12)('c') == 'c');

  • Byte

    Declaration

    alias Byte = SimpleTag!(byte, cast(NBT_TYPE)cast(ubyte)1u).SimpleTag;

    Tag with a signed byte, usually used to store small values like the progress of an action or the type of an entity. An unsigned version of the tag can be obtained doing a cast to ubyte. Byte Array is a tag with an array of unsigned bytes.

    Example:

    1. assert(cast(ubyte)(new Byte(-1)) == 255);

  • Bool

    Declaration

    alias Bool = SimpleTag!(byte, cast(NBT_TYPE)cast(ubyte)1u).SimpleTag;

    Byte tag that only uses the values 1 and 0 to indicate respectively true and false. It's usually used by SEL to store boolean values instead of a byte tag.

    Example:

    1. assert(new Byte(1) == new Bool(true));

  • Short

    Declaration

    alias Short = SimpleTag!(short, cast(NBT_TYPE)cast(ubyte)2u).SimpleTag;

    Tag with a signed short, used when the 255 bytes (or 127 if only the positive part is counted) is not enough. This tag can also be converted to its unsigned version doing a simple cast to ushort.

  • Int

    Declaration

    alias Int = SimpleTag!(int, cast(NBT_TYPE)cast(ubyte)3u).SimpleTag;

    Tag with a signed integer, used to store values that don't usually fit in the short tag, like entity's ids. This tag can aslo be converted to its unsigned version (uint) with a simple cast to it. Int Array is a tag with an array of signed integers.

  • Long

    Declaration

    alias Long = SimpleTag!(long, cast(NBT_TYPE)cast(ubyte)4u).SimpleTag;

    Tag with a signed long.

  • Float

    Declaration

    alias Float = SimpleTag!(float, cast(NBT_TYPE)cast(ubyte)5u).SimpleTag;

    Tag with a 4-bytes floating point value, usually used to store non-blocks coordinates or points in the world. The float.nan value can be used and recognized by the SEL-derived systems, but couldn't be recognized by other softwares based on different programming languages that doesn't support the not-a-number value. More informations about the NaN value and its encoding can be found on Wikipedia.

  • Double

    Declaration

    alias Double = SimpleTag!(double, cast(NBT_TYPE)cast(ubyte)6u).SimpleTag;

    Tag with an 8-bytes float point value used instead of the Float tag if the precision or the available number's range must be higher. See Float's documentation for informations about the NaN value and its support inside and outside SEL.

  • String

    Declaration

    alias String = SimpleTag!(string, cast(NBT_TYPE)cast(ubyte)8u).SimpleTag;

    Tag with an UTF-8 string encoded as its length as short and its content casted to btyes.

    Example:

    1. assert(new String("test") == ""); assert(new String("", "test") == "test");

  • ArrayTag

    Declaration

    class ArrayTag(T, NBT_TYPE _type): Tag;
    class NumericArrayTag(T, NBT_TYPE _type) if (isNumeric!T): ArrayTag!(T, _type);

    Simple tag with array-related functions.

    Example:

    1. assert(new ByteArray([2, 3, 4]).length == new IntArray([9, 0, 12]).length); auto b = new ByteArray(); assert(b.empty); b ~= 14; assert(b.length == 1 && b[0] == 14);

    • opOpAssign

      Declaration

      @safe void opOpAssign(string op : "~", G)(G value) if (is(G == T) || is(G == T[]));

      Concatenates T, an array of T or a NBT array of T to the tag.

      Example:

      1. auto array = new IntArray([1]); array ~= 1; assert(array == [1, 1]); array ~= [1, 2, 3]; assert(array == [1, 1, 1, 2, 3]); array ~= new IntArray([100, 99]); assert(array == [1, 1, 1, 2, 3, 100, 99]);

    • opBinary

      Declaration

      @safe typeof(this) opBinary(string op : "~", G)(G value) if (is(G == T) || is(G == T[]));

      Does the same job opOpAssign does, but creates a new instance of typeof(this) with the same name of the tag and returns it.

      Example:

      1. auto array = new IntArray([1, 2, 3]); assert(array ~ [2, 1] == [1, 2, 3, 2, 1] && array == [1, 2, 3]);

  • ByteArray

    Declaration

    alias ByteArray = NumericArrayTag!(byte, cast(NBT_TYPE)cast(ubyte)7u).NumericArrayTag;

    Array of unsigned bytes (clients and other softwares may interpret the bytes as signed due to limitations of the programming language). The tag is usually used by Minecraft's worlds to store blocks' ids and metas.

    Discussion

    If a signed byte is needed a cast operation can be done.

    Example:

    1. auto unsigned = new ByteArray([0, 1, 255]); auto signed = cast(byte[])unsigned; assert(signed == [0, 1, -1]);

  • IntArray

    Declaration

    alias IntArray = NumericArrayTag!(int, cast(NBT_TYPE)cast(ubyte)11u).NumericArrayTag;

    Array of signed integers, introduced in the last version of the NBT format. Used by anvil worlds.

    Discussion

    The same cast rules also apply for this tag's values.

    Example:

    1. auto signed = new IntArray([-1]); assert(cast(uint[])signed == [uint.max]);

    • get

      Declaration

      pure @safe T get(T : Tag)(string name, lazy T defaultValue);
      pure @safe T get(T : Tag, E)(string name, E defaultValue) if (!is(T == E) && __traits(compiles, new T(defaultValue)));

      Gets the element at the given index, if exists and can be casted to T. Otherwise evaluates and returns defaultValue.

      Return Value

      the named tag of type T or defaultValue if the conversion fails

      Example:

      1. auto compound = new Compound(new Named!String("test", "value")); assert(is(typeof(compound["test"]) == NamedTag)); assert(is(typeof(compound.get!String("test", null)) == String)); assert(compound.get!String("failed", new String("failed")) == new String("failed")); assert(compound.get!String("?", "string") == new String("string"));

    • getValue

      Example:

      1. auto compound = new Compound(new Named!String("test", "value")); assert(compound.getValue!String("test", "") == "value"); assert(compound.getValue!Int("test", 0) == 0); assert(compound.getValue!String("miss", "miss") == "miss");
