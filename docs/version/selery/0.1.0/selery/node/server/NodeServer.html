    Class NodeServer

    Singleton for the server instance.

    class NodeServer
      : EventListener!(selery.event.node.server.NodeServerEvent)
      , Server
      , sel.hncom.handler.HncomHandler!(clientbound) ;

    Properties

    NameTypeDescription
    args[get] const(shared(string[]))Gets the arguments the server has been launched with, excluding the ones used by sel or the manager.
    defaultWorld[get] shared(WorldInfo)Gets the server's default world.
    defaultWorld[set] shared(WorldInfo)Sets the server's default world if the given world has been created and registered using the addWorld template method.
    getWorldByName[set] shared(WorldInfo)Gets a world by its name.
    hubAddress[get] std.socket.AddressGets the address of the hub this node is connected to.
    hubLatency[get] uintGets the latency between the node and the hub.
    hubMax[get] ulongGets the number of maximum players that can connect to the hub. If the value is 0 it means that no limit has been set and players will never be kicked because the server is full.
    hubOnline[get] ulongGets the number of online players in the whole server (not just in the current node).
    id[get] immutable(long)Gets the server's id, which is equal in the hub and all connected nodes. It is generated by SEL's snooping system or randomly if the service cannot be reached.
    isMainNode[get] boolGets whether or not this is a main node (players are added when connected to hub) or not (player are added only when transferred by other nodes).
    max[get] ulongGets the maximum number of players that can connect on the current node (not in the whole server). If the value is 0 there's no limit.
    max[set] ulongSets the maximum number of players that can connect to the current node. 0 can be used for unlimited players.
    name[get] stringGets the server's name, as indicated in the hub's settings.txt file. The name should just the name of the server without any formatting code nor description.
    nodeName[get] stringGets the current's node name.
    nodes[get] const(Node)[]Gets a list with the nodes connected to the hub, this excluded.
    online[get] ulongGets the number of online players in the current node (not in the whole server).
    players[get] shared(PlayerInfo)[]Gets a list with all the players in the server.
    social[get] const(shared(selery.util.tuple.Tuple!(string,"website",string,"facebook",string,"twitter",string,"youtube",string,"instagram",string,"googlePlus")))Gets the server's social informations like website and social networks' names.
    uptime[get] core.time.DurationGets the server's uptime as a Duration instance with milliseconds precision. The count starts when the node is duccessfully connected with the hub.
    website[get] stringGets the server's website.
    worlds[get] shared(WorldInfo)[]Gets a list with every world registered in the server. The list is a copy of the one kept by the server and its modification has no effect on the server.
    config[get] const(Config)Gets the server's configuration.
    files[get] const(Files)Gets the server's files (assets and temp files).
    lang[get] const(LanguageManager)Gets the server's language manager.
    logger[get] selery.log.LoggerGets the server's logger.
    plugins[get] const(Plugin)[]Gets the plugins actived on the server.

    Methods

    NameDescription
    addWorld Creates and registers a world with its group, initialising its terrain, registering events, commands and tasks.
    broadcast Broadcasts a message to every node connected to the hub.
    broadcast Broadcasts a message in every registered world and their children calling the world's broadcast method.
    kick Registers a command.
    nodeWithHubId Gets a node by its hub id, which is given by the hub and unique for every session.
    nodeWithName Gets a node by its name. It can be used to transfer players to it.
    registerCommand Registers a command.
    removeWorld Removes a world and unloads it. When trying to remove the default world a message error will be displayed and the world will not be unloaded.
    sendMessage Sends a message to a node.
    shutdown Stops the server setting the running variable to false and kicks every player from the server.
    addEventListener Adds an event through a delegate.
    callCancellableIfExists Calls a cancellable event using callEventIfExists.
    callEvent Calls an event. Events are always called in the order they are registered, even in the inheritance.
    callEventIfExists Calls an event only if it exists. This should be used when a event is used only to notify the plugin of it.
    opOpAssign Adds an event through a delegate.
    opOpAssign Removes an event listener using its delegate pointer.
    opOpAssign Removes an event listener using its assigned id.
    removeEventListener Removes an event listener using its delegate pointer.
    removeEventListener Removes an event listener using its assigned id.
    updateLanguageFiles Updates langauge files (in config) and send the UpdateLanguageFiles packet to the hub if needed.
