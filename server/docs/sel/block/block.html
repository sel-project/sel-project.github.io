      Parameters

      the player who tapped the block

      Item item

      the item used, is the same as player.inventory.held

      BlockPosition position

      ubyte face

      the face tapped

      Return Value

      false is a block should be placed, true otherwise

    • breathe

      Declaration

      final bool breathe(World world, BlockPosition position, bool checkFluid = true);

      Boolean value indicating whether or not the upper block is air or isn't solid.

      Parameters

      World world

      the world there the block is placed

      BlockPosition position

      position in the world where the block is placed

      bool checkFluid

      boolean value indicating whether or not the fluid should be considered as a solid block

      Example:

      1. // farmlands become when dirt when they can't breathe world[0, 0, 0] = Blocks.FARMLAND; world[0, 1, 0] = Blocks.BEETROOT_BLOCK; assert(world[0, 0, 0] == Blocks.FARMLAND); world[0, 1, 0] = Blocks.DIRT; assert(world[0, 0, 0] != Blocks.FARMLAND);
