    Variable NodeInfo.additionalJSON

    Optional informations about the server's software and system, similar to HubInfo's additionalJson field.

    struct NodeInfo
    {
      // ...
      std.json.JSONValue additionalJSON ;
      // ...
    }

    Example

    {
  "software": {
     "name": "Selery",
     "version": "0.1.0",
     "stable": true
  },
  "system": {
     "os": "Windows 10",
     "cpu": "Intel(R) Core(TM) i7-5700U CPU @ 3.40GHz",
     "cores": 4,
     "ram": 8589934592
  }
}
