    Struct RemoteCommand

    Executes a command on the node.

    struct RemoteCommand ;

    Fields

    NameTypeDescription
    command stringCommands and arguments that should be executed on the node. For example say hello world or kill @a.
    commandId uintIdentifier of the command. It's sent back in Log's commandId field when the command generates output.
    origin ubyteOrigin of the command. It could be the hub itself or an external source.
    sender std.socket.AddressAddress of the sender if the command has been sent from an external source. It's null when the hub is the sender.
