  • Reload

    Declaration

    struct Reload;

    Notifies the node that the hub's reloadeable settings have been reloaded and that the node should also reload its resources (for example plugin's settings). The fields of the packet may be empty if not updated during the reload.

    • displayName

      Declaration

      string displayName;

      Display name of the server in the same format as HubInfo's displayName field.

    • motds

      Declaration

      string[ubyte] motds;

      New MOTDs (message of the day) for the supported games.

    • language

      Declaration

      string language;

      Main language of the server in the same format as HubInfo's language field.

    • acceptedLanguages

      Declaration

      string[] acceptedLanguages;

    • additionalJSON

      Declaration

      JSONValue additionalJSON;

  • Log

    Declaration

    struct Log;

    Sends a logged message to the hub.

    • timestamp

      Declaration

      ulong timestamp;

      Unix time (in milliseconds) that indicates the exact creation time of the log (for ordering purposes).

    • worldId

      Declaration

      int worldId;

      Id of the world that has generated the log, if the log comes from a world, -1 otherwise.

    • logger

      Declaration

      string logger;

      Name of the logger thas has generated the log, if the log doesn't come from a world.

    • message

      Declaration

      string message;

      Logged message. It may contain Minecraft formatting codes.

    • commandId

      Declaration

      int commandId;

      Identifier of the command that has generated the log or -1 if the log wasn't generated by a command.

  • AddNode

    Declaration

    • hubId

      uint hubId;

    • name

      Declaration

      string name;

      Node's name used for displaying and identification purposes.

    • main

      Declaration

      bool main;

      Whether the node is a main node.

    • acceptedGames

      Declaration

      uint[][ubyte] acceptedGames;

      Indicates the games and protocols accepted by the node.

  • RemoveNode

    Declaration

    struct RemoveNode;

    Notifies the node that another node, previously added with AddNode, has disconnected from the hub.

    • hubId

      Declaration

      uint hubId;

      Node's id given by the hub.

  • ReceiveMessage

    Declaration

    struct ReceiveMessage;

    Receives a binary message sent by another node using SendMessage.

    • sender

      Declaration

      uint sender;

      Id of the node that has sent the message.

    • broadcasted

      Declaration

      bool broadcasted;

      Indicates whether the message was broadcasted to every connected node.

    • payload

      Declaration

      ubyte[] payload;

      Bytes received. It could be serialised data or a plugin-defined packet.

  • SendMessage

    Declaration

    struct SendMessage;

    Sends a binary message to some selected nodes or broadcast it.

    • addressees

      Declaration

      uint[] addressees;

      Addressees of the message. If the array is empty the message is broadcasted to every connected node but the sender.

    • payload

      Declaration

      ubyte[] payload;

      Bytes to be sent/broadcasted. It may be serialised data or a plugin-defined packet.

  • UpdatePlayers

    Declaration

    struct UpdatePlayers;

    Updates the number of players on the server.

    • online

      Declaration

      uint online;

      Players currently online in the whole server (connected to a node).

    • max

      Declaration

      int max;

      Maximum number of players that can connect to server, which is the sum of the max players of every connected node.

  • UpdateUsage

    Declaration

    struct UpdateUsage;

    Updates the usage of the system's resources of the node.

    • ram

      Declaration

      uint ram;

      Kibibytes of RAM used by the node.

    • cpu

      Declaration

      float cpu;

      Percentage of CPU used by the node. It may be higher than 100 if the node has more than 1 CPU

  • RemoteCommand

    Declaration

    struct RemoteCommand;

    Executes a command on the node.

    • origin

      Declaration

      ubyte origin;

      Origin of the command. It could be the hub itself or an external source.

    • sender

      Declaration

      Address sender;

      Address of the sender if the command has been sent from an external source. It's null when the hub is the sender.

    • command

      Declaration

      string command;

      Commands and arguments that should be executed on the node. For example say hello world or kill @a.

    • commandId

      Declaration

      uint commandId;

      Identifier of the command. It's sent back in Log's commandId field when the command generates output.

  • AddWorld

    Declaration

    struct AddWorld;

    Notifies the hub that a new world has been created on the node.

    • worldId

      Declaration

      uint worldId;

      Id of the world. It's unique on the node.

    • name

      Declaration

      string name;

      Name of the world, it doesn't need to be unique.

    • dimension

      Declaration

      ubyte dimension;

      World's dimension in the MCPE format (0: overworld, 1: nether, 2: end).

    • parent

      Declaration

      int parent;

      Id of the world's parent or -1 if the world has no parent. This is usually used for nether/end which are children of an overworld world.

  • RemoveWorld

    Declaration

    struct RemoveWorld;

    Notifies the hub that a world has been removed from the node.

    • worldId

      Declaration

      uint worldId;

      Id of the world that has been removed, previosly added using the AddWorld packet.
