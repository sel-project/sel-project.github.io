    Class Block

    Base class for every block.

    class Block ;

    Properties

    NameTypeDescription
    bedrock[get] boolIndicates whether the block exists in Minecraft.
    blastResistance[get] doubleGets the blast resistance, used for calculate the resistance at the explosion of solid blocks.
    box[get] selery.math.vector.ClassicBlockBox!(double)If hasBoundingBox is true, returns the bounding box of the block as an Axis instance. Values are from 0 to 1
    data[get] sul.blocks.BlockGets the block's sul data.
    doRandomTick[get] boolBoolean value indicating whether or not the block can receive a random tick. This property is only requested when the block is placed.
    fallDamageModifier[get] floatModifies an entity's damage. The value should be higher than 0.
    flammable[get] boolBoolean value indicating whether or not the block can be burnt.
    fluid[get] boolIndicates whether the block is a fluid.
    hardness[get] doubleIndicates the block's hardness, used to calculate the mining time of the block's material.
    hasBoundingBox[get] boolIndicates whether the block has a bounding box which entities can collide with, even if the block is not solid.
    id[get] ushortGets the block's SEL id.
    indestructible[get] boolIndicates whether the block can be mined.
    instantBreaking[get] boolIndicates whether the block can be mined or it's destroyed simply by a left-click.
    java[get] boolIndicates whether the block exists in Minecraft.
    luminance[get] ubyteIndicates the level of light emitted by the block in a range from 0 to 15.
    opacity[get] ubyteGets the block's opacity, in a range from 0 to 15, where 0 means that the light propagates like in the air and 15 means that the light is totally blocked.
    replaceable[get] boolBoolean value indicating whether or not the block is replaced when touched with a placeable item.
    solid[get] boolIndicates whether a block is solid (can sustain another block or an entity) or not.

    Methods

    NameDescription
    breathe Boolean value indicating whether or not the upper block is air or isn't solid.
    drops Get the dropped items as a slot array.
    onEntityCollide Called when an entity collides with the block's side (except top).
    onEntityInside Called when an entity is inside the block (or part of it).
    onEntityStep Called when an entity falls on walks on the block.
    onInteract Function called when a player right-click the block. Blocks like tile should use this function for handle the interaction. N.B. That this function will no be called if the player shifts while performing the right-click/screen-tap.
    onRandomTick If the property doRandomTick is true, this function could be called undefined times duraing the chunk's random ticks.
    onScheduledUpdate Function called by the world after a requets made by the block using World.scheduleBlockUpdate if the rule in the world is activated.
    onUpdated Function called when the block is receives an update. Redstone mechanism should be handled from this function.
    opEquals Compare the block names.
    xp Get the amount of dropped xp when the block is broken
