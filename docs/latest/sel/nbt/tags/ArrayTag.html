    Class ArrayTag

    Simple tag with array-related functions.

    class ArrayTag(T, NBT_TYPE _type) ;

    Properties

    NameTypeDescription
    empty[get] boolChecks whether or not the array's length is equals to 0.

    Methods

    NameDescription
    opBinary Does the same job opOpAssign does, but creates a new instance of typeof(this) with the same name of the tag and returns it.
    opOpAssign Concatenates T, an array of T or a NBT array of T to the tag.
    remove Removes the element at the given index from the array.

    Example

    assert(new ByteArray([2, 3, 4]).length == new IntArray([9, 0, 12]).length);

auto b = new ByteArray();
assert(b.empty);
b ~= 14;
assert(b.length == 1 && b[0] == 14);
