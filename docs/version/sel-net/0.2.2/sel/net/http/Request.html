    Struct Request

    Container for a HTTP request.

    struct Request ;

    Fields

    NameTypeDescription
    data stringOptional raw form data, for POST requests.
    headers string[string]HTTP headers of the request.
    method stringMethod used in the request (i.e. GET). Must be uppercase.
    path stringPath of the request. It should start with a slash.
    valid boolIf the request was parsed, indicates whether it was in a valid HTTP format.

    Methods

    NameDescription
    get Creates a get request.
    parse Parses a string and returns a Request. If the request is successfully parsed Request.valid will be true. Please note that every key in the header is converted to lowercase for an easier search in the associative array.
    post Creates a post request.
    toString Encodes the request into a string.

    Example

    Request("GET", "/");
Request(Request.POST, "/subscribe.php");
