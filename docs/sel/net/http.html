    • post

      Declaration

      Creates a post request.

      Example:

      1. auto post = Request.post("/sub.php", ["Connection": "Keep-Alive"], "name=Mark&surname=White"); assert(post.toString() == "POST /sub.php HTTP/1.1\r\nConnection: Keep-Alive\r\n\r\nname=Mark&surname=White");

    • toString

      Declaration

      string toString();

      Encodes the request into a string.

      Example:

      1. auto request = Request(Request.GET, "index.html", ["Connection": "Keep-Alive"]); assert(request.toString() == "GET /index.html HTTP/1.1\r\nConnection: Keep-Alive\r\n");

    • parse

      Declaration

      static Request parse(string str);

      Parses a string and returns a Request. If the request is successfully parsed Request.valid will be true. Please note that every key in the header is converted to lowercase for an easier search in the associative array.

      Example:

      1. auto request = Request.parse("GET /index.html HTTP/1.1\r\nHost: 127.0.0.1\r\nConnection: Keep-Alive\r\n"); assert(request.valid); assert(request.method == Request.GET); assert(request.headers["Host"] == "127.0.0.1"); assert(request.headers["Connection"] == "Keep-Alive");

    • redirect

      Declaration

      static Response redirect(Status status, string location, string[string] headers = defaultHeaders);
      static Response redirect(uint statusCode, string location, string[string] headers = defaultHeaders);
      static Response redirect(string location, string[string] headers = defaultHeaders);

      Creates a 3xx redirect response and adds the Location field to the header. If not specified status code 301 Moved Permanently will be used.

      Example:

      1. Response.redirect("/index.html"); Response.redirect(302, "/view.php"); Response.redirect(StatusCodes.seeOther, "/icon.png", ["Server": "sel-net"]);

    • toString

      Declaration

      string toString();

      Encodes the response into a string. The Content-Length header field is created automatically based on the length of the content field.

      Example:

      1. auto response = Response(200, [], "Hi"); assert(response.toString() == "HTTP/1.1 200 OK\r\nContent-Length: 2\r\n\r\nHi");

    • parse

      Declaration

      static Response parse(string str);

      Parses a string and returns a Response. If the response is successfully parsed Response.valid will be true. Please note that every key in the header is converted to lowercase for an easier search in the associative array.

      Example:

      1. auto response = Response.parse("HTTP/1.1 200 OK\r\nContent-Type: plain/text\r\nContent-Length: 4\r\n\r\ntest"); assert(response.valid); assert(response.status == 200); assert(response.headers["content-type"] == "text/plain"); assert(response.headers["content-length"] == "4"); assert(response.content == "test");
