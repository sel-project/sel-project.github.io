    Aliases

    NameTypeDescription
    Bool SimpleTag!(byte,1) Byte tag that only uses the values 1 and 0 to indicate respectively true and false. It's usually used by SEL to store boolean values instead of a byte tag.
    Byte SimpleTag!(byte,1) Tag with a signed byte, usually used to store small values like the progress of an action or the type of an entity. An unsigned version of the tag can be obtained doing a cast to ubyte. Byte Array is a tag with an array of unsigned bytes.
    ByteArray NumericArrayTag!(byte,7) Array of unsigned bytes (clients and other softwares may interpret the bytes as signed due to limitations of the programming language). The tag is usually used by Minecraft's worlds to store blocks' ids and metas.
    Double SimpleTag!(double,6) Tag with an 8-bytes float point value used instead of the Float tag if the precision or the available number's range must be higher. See Float's documentation for informations about the NaN value and its support inside and outside SEL.
    Float SimpleTag!(float,5) Tag with a 4-bytes floating point value, usually used to store non-blocks coordinates or points in the world. The float.nan value can be used and recognized by the SEL-derived systems, but couldn't be recognized by other softwares based on different programming languages that doesn't support the not-a-number value. More informations about the NaN value and its encoding can be found on Wikipedia.
    Int SimpleTag!(int,3) Tag with a signed integer, used to store values that don't usually fit in the short tag, like entity's ids. This tag can aslo be converted to its unsigned version (uint) with a simple cast to it. Int Array is a tag with an array of signed integers.
    IntArray NumericArrayTag!(int,11) Array of signed integers, introduced in the last version of the NBT format. Used by anvil worlds.
    Long SimpleTag!(long,4) Tag with a signed long.
    LongArray NumericArrayTag!(long,12) Array of signed longs, introduced in Minecraft: Java Edition 1.13 and not yet in the other versions of Minecraft.
    Short SimpleTag!(short,2) Tag with a signed short, used when the 255 bytes (or 127 if only the positive part is counted) is not enough. This tag can also be converted to its unsigned version doing a simple cast to ushort.
    String SimpleTag!(string,8) Tag with an UTF-8 string encoded as its length as short and its content casted to btyes.
