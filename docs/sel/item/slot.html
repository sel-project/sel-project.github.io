sel.item.slot

  • Slot

    Declaration

    struct Slot;

    Container for an item stack that has always a value. If the slot is not empty, an Item instance and the count can be stored.

    Discussion

    Conventionally an empty slot is always constructed using Slot(null)

    • this

      Declaration

      @nogc @safe this(Item item);

      Creates a slot with an item and its highest value as count.

      Example:

      1. assert(Slot(new Items.DiamondSword()).count == 1); assert(Slot(new Items.Snowball()).count == 16); assert(Slot(null).count == 0);

    • this

      Declaration

      @nogc @safe this(Item item, ubyte count);

      Creates a slot giving an item and a count in range 0..255 (unsigned byte). Setting an higher value it's impossible due to protocol's limitations.

      Example:

      1. assert(Slot(new Items.DiamondSword(), 22).count == 22); assert(Slot(new Items.Snowball(), 2).count == 2); assert(Slot(null, 100).count == 0);

    • item

      Declaration

      pure nothrow @nogc @property @safe Item item();

      Gets the slot's item.

    • empty

      Declaration

      pure nothrow @nogc @property @safe bool empty();

      Checks whether or not the slot is empty. A slot is considered empty when its count is equal to 0 or when its item is null.

      Example:

      1. Slot slot = Slot(null); assert(slot.empty); slot = Slot(new Items.Snowball(), 1); assert(!slot.empty); slot.count--; assert(slot.empty);

    • full

      Declaration

      pure nothrow @nogc @property @safe bool full();

      Checks whether or not the slot is full. The slot is considered full when its count is equals or higher than the item's max stackable size.

      Example:

      1. assert(Slot(new Items.DiamondSword(), 1).full); assert(!Slot(new Items.Snowball(), 15).full);


      This property should be called only if the slot isn't empty due to its call on the item's property, that can be null.
      1. if(!slot.empty && slot.full) { ... }

    • fill

      Declaration

      pure nothrow @nogc @safe void fill();

      Fills the slot setting the count as the max stacking value of the item.

      Example:

      1. Slot slot = Slot(new Item.Snowball(), 1); assert(slot.count == 1); slot.fill(); assert(slot.count == 16);


      Like full, this function requires item to not be null due to its call on its max property.
      1. if(slot.item !is null) { slot.fill(); }
