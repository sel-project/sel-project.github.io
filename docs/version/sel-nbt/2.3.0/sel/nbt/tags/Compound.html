    Class Compound

    Associative array of named tags (that can be of different types).

    class Compound
      : Tag ;

    Properties

    NameTypeDescription
    dup[get] CompoundCreates an exact duplicate of the tag.
    empty[get] boolChecks whether or not the array is empty (its length is equal to 0).
    length[get] ulongGets the length of the array (or the number of NamedTags in it).
    names[get] string[]Gets the keys (indexes of the array).
    type[get] NBT_TYPEGets the tag's type.

    Methods

    NameDescription
    get Gets the element at the given index, if exists and can be casted to T. Otherwise evaluates and returns defaultValue.
    getValue Gets the tag's value at the given index, if it exists and can be casted to T. Otherwise returns defaultValue.
    has Checks whether or not a value is in the associative array.
    has Checks if the key is associated to a value and that the value is of the same type of T.
    opBinaryRight Gets a pointer to the element at the given index.
    opIndex Gets the array of named tags (without the keys). To get the associative array of named tags use the property value.
    opIndex Gets the element at the given index.
    opIndexAssign Sets the value at the given index. If the tag's name is different from the given index, the tag's name will be changed to the given index's one.
    opIndexAssign Sets the value using the named tag's name as the index.
    remove Removed the given index from the array, if set.
    decode Decodes the tag's body.
    encode Encodes the tag's body.
    name Gets the tag's name, if there's one.
    named Indicates whether the tag has a name.
    rename Creates a NamedTag maintaing the tag's properties.
    toJSON Encodes the tag's value as json.
    toString Encodes the tag a human-readable string.

    Example

    auto compound = new Compound();
compound["string"] = new String("test");
compound["byte"] = new Byte(18);
