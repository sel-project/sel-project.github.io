    Class Player

    Abstract class with abstract packet-related functions. It's implemented as another class by every version of Minecraft.

    Methods

    NameDescription
    callCommand Calls a command from a string.
    clearTitle Removes the title displayed with the title property.
    disconnect Disconnects the player from the server (from both the node and the hub). The reason can be a Translation.
    grantPermission Grants a permission.
    hasPermission Indicates whether the player has the given permission.
    hideTitle Hides the title displayed with the title property without resetting it.
    registerCommand Adds a new command using a command-container class.
    revokePermission Revokes a permission.
    sendTip Sends a tip message that will be displayed above the hotbar for two seconds before fading out.
    sendTitle Sends a title message that will be displayed at the centre of the screen. The Title struct can be used to control the title message, the subtitle and the timing for the animations (fade in, stay and fade out).
    transfer Transfers the player in another node. The target node should be in server.nodes, otherwise the player will be disconnected by the hub with "End of Stream" message.
    transfer Transfers a player to given server and port if the client has the functionality to do so. Calling this method will not disconnect the player immediately.
    unregisterCommand Removes a command using the command class given in registerCommand.
    sendMessage Sends a message to the command sender. The message can contain formatting codes and translations.
    handleCompleteMessage Completes the command args (or the command itself) if the arg type is an enum or a player (the ones in the world's list are sent), even if they are not spawned or visible to the player.
