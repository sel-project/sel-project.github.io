    Methods

    NameDescription
    attack Attacks an entity and returns the event used.
    drop Drop an item from this entity
    hide Hides an entity from this entity, if this entity can see it.
    move Moves the entity.
    sees Checks wether this entity can see or not another entity.
    setSize Updates the size of the entity and its bounding box.
    shouldSee Indicates which entities this one should and should not see. By default only the entities indicated as true by this function will be shown through the show function and added to the watchlist. For example, an arrow can see a painting, so Arrow.shouldSee(painting) will be true, but a painting shouldn't see an arrow, so Painting.shouldSee(arrow) will be false. This increases the performances as there are less controls, casts and operation on arrays in big worlds or chunks of worlds with an high concetration of entities.
    show Adds an entity to the ones this entity can see.
    teleport Teleports the entity.
    addEventListener Adds an event through a delegate.
    callCancellableIfExists Calls a cancellable event using callEventIfExists.
    callEvent Calls an event. Events are always called in the order they are registered, even in the inheritance.
    callEventIfExists Calls an event only if it exists. This should be used when a event is used only to notify the plugin of it.
