    • callEvent

      Declaration

      void callEvent(T : O)(ref T event) if (is(T == class) && !isAbstractClass!T);

      Calls an event. Events are always called in the order they are registered, even in the inheritance.

    • callEventIfExists

      Declaration

      T callEventIfExists(T : O)(lazy Parameters!(T.__ctor) args) if (is(T == class) && !isAbstractClass!T);

      Calls an event only if it exists. This should be used when a event is used only to notify the plugin of it.

      Return Value

      the instance of the event or null if the event hasn't been called

    • callCancellableIfExists

      Declaration

      bool callCancellableIfExists(T : O)(lazy Parameters!(T.__ctor) args) if (is(T == class) && !isAbstractClass!T && is(T : Cancellable));

      Calls a cancellable event using callEventIfExists.

      Return Value

      true if the event has been called and cancelled, false otherwise

  • Event

    Declaration

    interface Event;

    Base interface of the event. Every valid event instance (that is not an interface) must implement this interface.

  • Cancellable

    Declaration

    interface Cancellable;
    alias Cancelable = Cancellable;

    Indicates that the event is cancellable and its propagation can be stopped by plugins.

    • cancel

      Declaration

      abstract pure nothrow @nogc @safe void cancel();

      Cancels the event. A cancelled event is not propagated further to the next listeners, if there's any.

      Example:

      1. example += (ExampleEvent event){ log("1"); }; example += (ExampleEvent event){ log("2"); event.cancel(); }; example += (ExampleEvent event){ log("3"); }; assert(event.callCancellableIfExists!ExampleEvent());
      The example will print
      1. 1 2

    • cancelled

      Declaration

      abstract pure nothrow @nogc @property @safe bool cancelled();
      alias canceled = cancelled;

      Indicates whether the event has been cancelled. A cancelled event cannot be uncancelled.
