    Function Compound.opIndex

    Gets the array of named tags (without the keys). To get the associative array of named tags use the property value.

    Tag[] opIndex () pure nothrow @safe;

    Example

    Compound compound = new Compound([new Byte(1), new Int(2)]);
assert(compound[] == compound.value.values);

    Function Compound.opIndex

    Gets the element at the given index.

    Tag opIndex (
      string name
    ) pure nothrow @safe;

    Throws

    RangeError if the given index is not in the array

    Example

    assert(new Compound(new Named!String("0", "test"))["0"] == "test");
