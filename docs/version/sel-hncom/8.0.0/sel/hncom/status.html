    Module sel.hncom.status

    Structs

    NameDescription
    AddNode Notifies the node that another node (that is not the receiver) has connected to the hub.
    AddWorld Notifies the hub that a new world has been created on the node.
    Latency Used to calculate latency by both the hub and the node. When this packet is received it should be immeditaly sent back to the sender without any change.
    Log Sends a logged message to the hub.
    ReceiveMessage Receives a binary message sent by another node using SendMessage.
    RemoteCommand Executes a command on the node.
    RemoveNode Notifies the node that another node, previously added with AddNode, has disconnected from the hub.
    RemoveWorld Notifies the hub that a world has been removed from the node.
    SendMessage Sends a binary message to some selected nodes or broadcast it.
    UpdateMaxPlayers Updates the number of players that can be accepted by the node. If the given number is smaller than the players currently connected to the node no player should be kicked.
    UpdatePlayers Updates the number of players on the server.
    UpdateUsage Updates the usage of the system's resources of the node.
    WebAdminCredentials Gives the node the credentials that will be used by a remote panel client to connect to a world.
