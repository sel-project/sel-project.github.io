    Struct Response

    Container for an HTTP response.

    struct Response ;

    Fields

    NameTypeDescription
    content stringContent of the request. Its type should be specified in the content-type field in the headers.
    headers string[string]HTTP headers of the request.
    status StatusStatus of the response.
    valid boolIf the response was parsed, indicates whether it was in a valid HTTP format.

    Methods

    NameDescription
    error Creates a response for an HTTP error an automatically generates an HTML page to display it.
    parse Parses a string and returns a Response. If the response is successfully parsed Response.valid will be true. Please note that every key in the header is converted to lowercase for an easier search in the associative array.
    redirect Creates a 3xx redirect response and adds the Location field to the header. If not specified status code 301 Moved Permanently will be used.
    toString Encodes the response into a string. The Content-Length header field is created automatically based on the length of the content field.

    Example

    Response(200, ["Connection": "Close"], "<b>Hi there</b>");
Response(404, [], "Cannot find the specified path");
Response(204);
