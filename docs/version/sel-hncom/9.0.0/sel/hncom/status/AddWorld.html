    Struct AddWorld

    Notifies the hub that a new world has been created on the node.

    struct AddWorld ;

    Fields

    NameTypeDescription
    default_ boolIndicates whether the world is the node's default (the one where new players are spawned into if not handled by plugins). If the node does not support default worlds or it simply doesn't have one this field should be left to its default value (false). If the hub receives more than one default world it should consider the last received one as the default world.
    dimension ubyteWorld's dimension in the MCPE format (0: overworld, 1: nether, 2: end).
    groupId uintWorld's group id. Similar worlds can be grouped together and removed with the RemoveWorldGroup packet. If the world is independent and has no group the group id must be the same as the world id.
    name stringName of the world, it doesn't need to be unique.
    worldId uintId of the world. Must be unique on the node.
