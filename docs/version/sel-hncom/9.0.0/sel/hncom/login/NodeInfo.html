    Fields

    NameTypeDescription
    acceptedGames uint[][ubyte]Informations about the games accepted by the node. There should be at least one combination of game/protocols that is also accepted by hub as indicated in HubInfo.gamesInfo, otherwise the node will never receive any player.
    additionalJSON std.json.JSONValueOptional informations about the server's software and system, similar to HubInfo's additionalJson field.
    max uintMaximum number of players accepted by node.
