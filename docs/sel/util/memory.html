sel.util.memory

  • bytes

    Declaration

    @property @safe ulong bytes(string value);

    Gets the memory in bytes from a string.

    Parameters

    string value

    a value formatted as "float type"

    Return Value

    the given value in bytes, or 0 if bad formatting

    Example:

    1. assert("100 kB".bytes == 100000); assert("1000 MB".bytes == "1 GB".bytes);

  • usage

    Declaration

    @property @safe string usage(double mem, bool bin = false);

    Formats the given memory with the best unity of measurement.

    Parameters

    double mem

    the memory to be formatted

    bool bin

    boolean value indicating whether or not to use powers of 2s instead of power of 10s

    Return Value

    the formatted string

    Example:

    1. assert(usage(1024, false) == "1.024 kB"); d(usage(memory));

  • Memory

    Declaration

    struct Memory;

    Memory stored as bytes and available in various formats.

    Example:

    1. assert(Memory(1000).kilobytes == 1); assert(Memory("1 MB").megabytes != Memory("1 MB").mibibytes);
